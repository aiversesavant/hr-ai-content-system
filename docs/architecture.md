# Architecture

## Purpose
HR AI Content System is an AI-assisted content pipeline project for HR-related document processing and retrieval workflows.

## Core Entry Point
- `app.py` — main application entry point

## Pipeline Components
Pipeline logic is organized under `pipeline/`, including:
- chunking
- embeddings
- evaluation
- governance
- ingestion
- metadata
- retrieval

## Data Area
- `data/` — local project data and sample documents

## Notes
This project is being normalized to the enterprise workspace standard while preserving current behavior.
